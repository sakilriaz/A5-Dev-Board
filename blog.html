Question-1: What are the different ways to select an element in the DOM?

We can select an element in the DOM by using getElementByID, getElementsByClass,getElementsByTagName, querySelector and querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

When we use innerHTML in javascript it gets or sets elements, tags and attributes. If we use innerText it will provide only the visible text but when we use textContent it provide all text even those display are hidden by css property.

What is event delegation in the DOM?

Event delegation is related to event bubbling. When an event occurs on a child elements it bubbles up to its parent,grandparent upto the root.So, instead of adding multiple event listeners in multiple child element here the event listener is placed in upstream to listen for downstream. Example- adding event listener in OL/UL to control Li.

Question-4: What is event bubbling in the DOM?

When an event occurs on a element it propagates from that target element to its parent,grandparent upto the root. This propagation process in DOM is known as bubbling. Example- if we click a button inside a "div", the click event first triggers on the button, then bubbles up to its parent "div", then to "body"and so on.

How do you create, add, and remove elements using JavaScript?

We use "document.createElement()" to create an element in javascript. We use "appendChild()" to add and "removeChild()" to remove an element in javascript.
